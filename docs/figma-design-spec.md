Based on the screenshots, I would keep the same visual language:

### Design Style Guide

**Theme:** Modern AgriTech Dashboard
**Primary Color:** #2FB344 (Green)
**Secondary Gradient:** #2FB344 → #A3E635 (Lime Green)
**Accent:** #22C55E
**Background:** #F5F7FA
**Cards:** White (#FFFFFF) with subtle shadows
**Success:** #10B981
**Warning:** #F59E0B
**Danger:** #EF4444
**Typography:**

* Heading: Poppins Bold
* Body: Inter / Roboto
* Rounded corners: 12px–16px
* Large dashboard cards
* Simple icon-based navigation
